Audio-List
---
A simple web component for playlists and audio-elements.

[Demo](http://mp3.w8l.org/hp1/playlst/)

Attributes
---

|attr|type|default|
|---|---|---|
|autoplay|bool|true|
|loop|bool|true|
|shuffle|bool|false|
|src|string|./playlist.json|
|callback|string|playlist|

Example
---
`<play-list src="playlist.json" shuffle="true"></play-list>`
